# COMP1844 - Information Analysis and Visualisation

## Laboratory Session 3: Data Visualisation

### Overview

This lab session focuses on **visualizing one-dimensional data** using NumPy arrays and matplotlib. Students will work with temperature data from two cities with different climates to create various chart types.

---

## Prerequisites

- Python 3.x
- NumPy library
- Matplotlib library
- Completion of Lab 1 and Lab 2

---

## Tasks

### Task 1: Data Table

Create and display a table with three columns:

1. Month name
2. Average monthly temperatures of the first city
3. Average monthly temperatures of the second city

### Task 2: Data Visualisation

Visualize the average monthly temperatures using:

- **2.1** Line Chart
- **2.2** Bar Chart
- **2.3** Box Plot Chart

---

## Getting Started

1. Identify two cities with different climates
2. Initialize two NumPy arrays with average monthly temperatures for each city
3. Use code fragments from lecture slides to complete the tasks
