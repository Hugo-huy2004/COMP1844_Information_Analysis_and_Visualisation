# Coursework Report

## 1. Overview
This project is a coursework submission for the COMP1844 Information Analysis and Visualisation module. The program provides a simple web-based tool for converting distances between kilometers and miles, visualizing the results, and summarizing the data in a user-friendly format. The coursework demonstrates basic data collection, processing, and visualization skills using web technologies.
